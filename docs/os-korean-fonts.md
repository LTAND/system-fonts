# 操作系统内置的韩文字体

## Windows

- Batang (韩语：바탕) 类似中文字体的楷体
- BatangChe (韩语：바탕체)
- Gungsuh (韩语：궁서) 宫书体，一种手写体，相当于中文字体中的楷体或行楷体
- GungsuhChe (韩语：궁서체)
- Gulim (韩语：굴림) 对应中文字体的圆体
- GulimChe (韩语：굴림체)
- Dotum (韩语：돋움) 对应中文字体的黑体
- DotumChe (韩语：돋움체)
- Malgun Gothic (韩语：맑은고딕)，在 Windows Vista 时预装

### macOS

- Apple SD Gothic Neo
- AppleGothic
- AppleMyungjo
- Nanum Gothic
- PCMyungjo
